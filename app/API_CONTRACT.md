# EcoSphere App ŌĆö API Contract (v1)

Standalone build of the EcoSphere spec (PLAN.md + ARCHITECTURE.md + wireframe/index.html),
Python FastAPI backend + React frontend. **This file is the binding contract between the two.**

- Base URL: `http://localhost:8000/api`
- Auth: `POST /auth/login` returns a bearer token. Frontend keeps it **in memory only**
  (never localStorage/sessionStorage) and sends `Authorization: Bearer <token>`.
- Errors: JSON `{"detail": "human-readable message"}` with 400/401/403/404/409/422/429.
  The server rejects invalid input; the UI merely explains it.
- Roles: `employee`, `admin` (org-scoped), `super` (platform tier ŌĆö Odoo-level admin,
  sees all organizations). Every route requires auth unless noted. `(admin)` = admin-only.
  **Org scoping:** admins/employees only ever see their own organization's records.
  The super role can call ONLY `/auth/*`, `/dashboard`, `/notifications` and `/platform/*`
  ŌĆö every org-scoped route returns 403 for it.
- Dates: ISO `YYYY-MM-DD`. Money-free; scores are 0ŌĆō100 floats rounded to ints in UI.
